Marketplace module¶

Модуль marketplace

Содержит базовые классы для работы с товарами и корзиной: - Product — товар - Cart — корзина покупателя

class marketplace.Cart¶

Базовые классы: object

Корзина покупателя.

add_product(product: Product) → None¶

Добавить товар в корзину.

Если товар уже есть — увеличивает количество.

Параметры:

product – Товар.

decrease_quantity(product_name: str) → None¶

Уменьшить количество товара.

Если количество станет 0 — товар удаляется.

Параметры:

product_name – Название товара.

increase_quantity(product_name: str) → None¶

Увеличить количество товара.

Параметры:

product_name – Название товара.

remove_product(product_name: str) → None¶

Удалить товар из корзины полностью.

Параметры:

product_name – Название товара.

total_sum() → float¶

Общая сумма корзины.
